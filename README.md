## Projects

### 1. SEO記事自動生成システム
> OpenAI GPT-4o + Google Cloud Run による業務自動化

- **概要**: SEO記事のキーワード分析から構成案・初稿生成までを自動化
- **技術**: Python, Cloud Run, OpenAI API, Google Sheets/Docs API, Search Console API
- **成果**:
  - 1記事あたりの作成時間を **3時間 → 30分** に短縮（80%削減）
  - 月間 **50記事以上** を自動生成
- **リポジトリ**: [article-automation](https://github.com/0604yamato/article-automation)

### 2. 歯科クリニック向けAIチャットボット
> Difyを活用した顧客対応の自動化

- **概要**: 歯科クリニックの公式サイトにAIチャットボットを導入
- **役割**: 要件定義、プロンプト設計、ナレッジベース構築、Web業者との技術連携
- **工夫点**:
  - 予約導線の最適化でコンバージョン率向上
  - Firecrawlを使ったナレッジ自動取り込み
